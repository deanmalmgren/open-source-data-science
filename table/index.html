Open source data science

Although tools are far from the most important consideration in a data science project, tools are a necessary component of any data science work. This project is an attempt, however feeble, to identify and track a suite of open source packages that can be used for data science projects, from data acquisition and storage all the way through the analysis and visualization. Unlike the map which shows how all of these macroscopic pieces fit together, the goal of this page is to provide a quick searchable guide that makes it easy to find particular packages, how actively maintained they are (a key component when selecting an open source tool), and to understsand their licensing requirements.
